# Modelos de Banco de Dados – MySQL

Este repositório reúne **modelos relacionais de banco de dados** desenvolvidos em **MySQL**, utilizando o **MySQL Workbench (Forward Engineering)**.  
Os projetos foram criados com fins **acadêmicos e didáticos**, abordando diferentes cenários de negócio e aplicando conceitos de modelagem relacional.

---

## Objetivo do Repositório

O objetivo deste repositório é:
- Praticar **modelagem de banco de dados relacional**
- Aplicar conceitos de **entidades, relacionamentos e integridade referencial**
- Organizar diferentes **estudos de caso** em um único repositório
- Servir como **material de estudo e portfólio técnico**

---
